## Reading Recommendations by Interest

### For Beginners

1. *Apology* (Plato) - Socrates' defense
2. *Nicomachean Ethics* I, X (Aristotle) - Happiness and contemplation
3. *Enchiridion* (Epictetus) - Practical Stoic wisdom
4. *Letter to Menoeceus* (Epicurus) - Epicurean ethics

### For Ethics

1. *Republic* (Plato) - Justice and ideal state
2. *Gorgias* (Plato) - Rhetoric vs. philosophy
3. *Nicomachean Ethics* (Aristotle) - Virtue ethics
4. *Meditations* (Marcus Aurelius) - Applied Stoicism

### For Metaphysics

1. *Phaedo* (Plato) - Forms and soul
2. *Parmenides* (Plato) - Problems with Forms
3. *Metaphysics* VII-IX (Aristotle) - Substance
4. *Enneads* V.1 (Plotinus) - The One

### For Epistemology

1. *Meno* (Plato) - Learning as recollection
2. *Theaetetus* (Plato) - What is knowledge?
3. *Posterior Analytics* (Aristotle) - Scientific demonstration
4. *Outlines of Pyrrhonism* (Sextus) - Skeptical method

### For Logic

1. *Prior Analytics* I.1-7 (Aristotle) - Syllogistic
2. *Sophistical Refutations* (Aristotle) - Fallacies
3. Stoic logic (via Long & Sedley) - Propositional logic
